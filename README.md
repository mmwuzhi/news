# news

A daily news digest that pulls from RSS feeds, summarizes each article in English and Chinese using Gemini, and publishes a static page via Cloudflare Pages.

Runs automatically at 15:00 Beijing time every day via GitHub Actions.

## How it works

1. `fetch.py` pulls up to 5 articles from each feed (40 total)
2. All articles are sent to Gemini in a single prompt — each gets a 2-sentence EN summary, a 2-sentence ZH summary, a Chinese headline translation, keyword tags, and a category label (AI / TECH / FINA / SCI / WORLD)
3. The script generates `index.html` and saves an archive snapshot to `archive/YYYY-MM-DD.html`
4. GitHub Actions commits the output and pushes; Cloudflare Pages deploys automatically

## Feeds

少数派, V2EX, Hacker News, OpenAI Blog, AI News, BBC, The Verge, MIT Tech Review, 36氪, MarketWatch, Seeking Alpha, GitHub Blog, New Scientist

## UI

Static HTML generated by Python, interactive via vanilla JS — no framework, no build step.

**Desktop (≥ 860px):** 960 × 680px terminal-style window with three independent scroll columns — category filter sidebar, article list, and archive / preferences sidebar.

**Mobile (< 860px):** Full-viewport layout with a horizontal category tab bar and slide-up bottom sheets for archive and preferences.

**Features:**
- Expand / collapse articles with a smooth animation
- ↑ / ↓ vote buttons per article, persisted in `localStorage`
- Inferred preferences panel — derives tag affinities from votes (BOOST / FILTER / EXPLORE)
- Load more (shows 8 articles initially, +4 per click)
- Dark / light theme toggle, default dark, persisted in `localStorage`
- Archive panel listing the last 14 daily snapshots

## Setup

**Secret required (GitHub → Settings → Secrets):**

- `GEMINI_API_KEY` — Google AI Studio key with Gemini API access

**Run locally:**

```sh
pip install -r requirements.txt
GEMINI_API_KEY=your_key python fetch.py
```

## Deployment

Connected to Cloudflare Pages — any push to `main` deploys automatically. Custom domain: `news.wuwuwu.cc`.

To trigger a manual run: GitHub → Actions → Daily News → Run workflow.
