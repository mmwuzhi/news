# news

A daily news digest that pulls from RSS feeds, summarizes each article in English and Chinese using Gemini, and publishes a static page via Cloudflare Pages.

Runs automatically at 15:00 Beijing time every day via GitHub Actions.

## How it works

1. `fetch.py` pulls up to 5 articles from each of the 13 feeds, sorts by recency, and keeps the 40 newest
2. If Supabase is configured, recent votes are read and turned into tag-affinity scores used to rank stories within each category
3. All articles are sent to Gemini in a single prompt â€” each gets a Chinese headline translation, a 2-sentence EN summary, a 2-sentence ZH summary, keyword tags, and a category label (AI / TECH / FINA / SCI / WORLD). If the primary model is overloaded, the script fails over to a backup model
4. Stories are grouped by category and capped at 8 per category
5. The script renders `index.html` from `template.html` (+ `assets/style.css`, `assets/app.js`), writes an archive snapshot to `archive/YYYY-MM-DD.html`, and updates the rolling `feed.xml` (RSS 2.0) and `feed.json` (JSON Feed 1.1) â€” last 7 days, max 200 items
6. GitHub Actions commits the output and pushes; Cloudflare Pages deploys automatically

## Setup

**Secrets (GitHub â†’ Settings â†’ Secrets):**

- `GEMINI_API_KEY` â€” Google AI Studio key with Gemini API access (required)
- `SUPABASE_URL`, `SUPABASE_KEY` â€” optional; enable cross-device vote sync and vote-weighted ranking. Without them, voting still works locally per browser

- `GEMINI_RETRIES` â€” attempts per model before failing over (default `4`)
